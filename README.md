# back_curso_mythos

API de cadastro do Curso Mythos

# Populand tabelas

Entre no banco mySql com seu usuário e senha e execute as querys do arquivo table.sql

# subindo serviço da API

- Uma vez clonado o repositório, no propt de comando, execute `yarn install` no diretório clonado para instalar todas as dependências
- No arquivo index.js, substitua os campos [usuario_banco] e [senha_banco] por seu usuário e senha cadastrados no seu banco mySql;

- No propt de comando, no diretório onde você clonou o repo do Back End, execute `yarn dev`
- Quando o serviço estiver disponível, a seguinte mensagem irá aparecer: `rodando na porta 3001`
